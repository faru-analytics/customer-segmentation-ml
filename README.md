# Customer Segmentation using K-Means

## Overview
This project uses K-Means clustering to segment customers based on their annual income and spending score.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Key Steps
- Data cleaning (handled missing values)
- Data visualization
- Elbow method to find optimal clusters
- Applied K-Means clustering

## Insights
- Identified 5 customer segments
- High income-high spending customers are premium targets
- High income-low spending customers can be converted
- Different groups require different marketing strategies

## Conclusion
Customer segmentation helps businesses understand their audience and improve targeted marketing strategies.
